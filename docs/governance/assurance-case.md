# Assurance Case

## Claim

Flux Behaviour can support public-service improvement without collecting personal content when consent, minimisation, explainability, oversight and release controls are enforced.

## Current evidence

Repository contract, GitHub desired settings, security policy, CODEOWNERS, conformance matrix, gap register, harm register, accessibility starter record and migration inventory exist.

## Current blockers

Live GitHub settings are not API-verified. Runtime collector is not promoted. DPIA is not complete. Accessibility testing is not complete. Release gate evidence is not complete.
